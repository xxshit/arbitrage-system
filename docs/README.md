# ArbiScope 项目知识库

此目录是项目的长期知识库，用于沉淀业务思路、技术设计、风险规则和决策记录。每次讨论形成可执行结论后，应更新对应文档。

| 文档 | 用途 |
| --- | --- |
| [架构设计](架构设计.md) | 系统结构、数据流、数据库与部署方式 |
| [套利策略知识](套利策略知识.md) | 策略定义、收益计算、风控要点 |
| [决策记录](决策记录.md) | 已作出的关键选择及其背景 |
| [开发路线图](开发路线图.md) | 已完成内容与下一阶段计划 |

## 维护规则

1. 新的业务规则、策略假设和约束写入“套利策略知识”。
2. 架构或技术方案变动写入“架构设计”和“决策记录”。
3. 每项功能的完成状态写入“开发路线图”。
4. 严禁将 API Key、私钥、交易账户密码或助记词写入任何文档或代码库。
