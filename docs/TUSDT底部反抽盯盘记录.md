# T/USDT 底部反抽盯盘记录

更新时间：2026-07-18

## 用户当前剧本

T 当前长时间在 `0.0040-0.0041` 附近横盘，前面几次涨跌后也大致回到这个区域。用户认为这里像阶段底部，后续可能先走一段拉高，再重新跌回 `0.0040` 左右，甚至更低。

用户核心判断：

- 主力可能在反复上涨出货并诱多。
- 当前如果出现向上反转异动，用户希望跟随做一段短线多。
- 上涨停滞后，用户希望平多，并观察反手做空。
- 大结构上，用户仍认为 T 后续下跌概率更高。

## 后台盯盘触发

系统对 T 单独加入 5 分钟级短线盯盘，不只看 30MIN。

### 阶段一：底部反抽做多提醒

触发条件：

- 当前价格仍在 `0.00392-0.00418` 附近，属于底部箱体范围；
- 价格站上 `0.00412`；
- 5MIN 价格涨幅大于约 `0.45%`；
- 5MIN 与 15MIN 的 CVD 均转强；
- 5MIN / 15MIN / 30MIN 任一窗口出现明显放量，放量倍数大于约 `1.35x`；
- 持仓没有快速塌掉；
- 多空人数比没有严重过热。

推送含义：

这不是趋势多，而是按“底部横盘后的短线反抽”处理。收到提醒后，用户可重点观察是否跟随做多。

### 阶段二：反抽停滞，平多/反手空观察

触发条件：

- 价格已经反抽到 `0.00428` 以上；
- 5MIN 与 15MIN 价格推进变慢；
- 出现 CVD 转弱，或多空人数比回升；
- 同时存在放量、负资费、负基差中的至少一个压力信号。

推送含义：

如果反抽已经走出，但放量不涨、主动买入变弱、多头账户变多，且负资费/负基差没有修复，则更符合“诱多后继续下跌”的路径。此时优先提示平多，并开始观察反手空。

## 关键区间

- 底部箱体：`0.0040-0.0041`
- 启动确认：`0.00412` 上方
- 第一反抽观察区：`0.00428-0.00438`
- 停滞/反手观察区：`0.00428-0.00445`
- 空头逻辑降级：放量站稳 `0.0047` 上方，并且基差、资费明显修复

## 复盘规则

- 如果 T 后续按“先拉高、再跌回 0.0040 甚至更低”的路径运行，记录为用户对主力诱多/反复出货路径判断有效。
- 如果 T 放量站稳 `0.0047` 上方，并且负基差、负资费修复，则要检讨“过早预设出货”的偏差。
- 这条剧本必须和爆仓数据一起验证：如果持仓或多空人数比突然断裂，需要优先检查是否是集中爆仓造成，而不是直接解释成主动资金行为。
