# T/USDT 盯盘推送规则

更新时间：2026-07-20

## 当前用户要求

- T 如果只是延续下跌，暂时不要继续推送；
- 后期如果出现反转向上的趋势，要推送；
- 如果先出现反抽上涨，后续又在关键区域停滞转弱，也可以推送“平多 / 反手空观察”；
- 推送重点必须是“新结构、新走势、新判断”，不是重复说明它还在跌。

## 执行规则

T 不再走通用 `bearish_db_watch` 兜底逻辑，只走 T 专属剧本：

1. `t_bounce_long`
   - 底部区域附近；
   - 价格向上突破短线箱体；
   - 5MIN / 15MIN CVD 转强；
   - 有放量；
   - 持仓没有快速塌掉；
   - 多空人数比没有严重过热。

2. `t_bounce_stall_short`
   - 价格已经反抽到关键区域；
   - 推进开始变慢；
   - CVD 转弱，或多空人数比回升；
   - 同时存在放量、负资费、负基差中的至少一个压力信号。

## 去重规则

T 的推送 key 按价格区间识别，不再按极细价格跳动识别：

- `floor-watch`
- `floor-break-412-428`
- `bounce-428-445`
- `stall-445-470`
- `above-470`

这样可以避免 T 因为几跳价格变化就重复推送。
