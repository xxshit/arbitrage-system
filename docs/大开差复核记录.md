# 大开差复核记录

更新时间：2026-07-20

## 复核规则

- 复核范围：当前行情快照中 `abs(开差) > 10%` 的币种。
- 优先检查是否存在“同名不同币”：项目名称、官网、链、合约地址是否一致。
- 如果不是同名不同币，也要继续检查是否存在充提关闭、公告下架、价格保护、现货指数异常、盘口深度过薄等情况。
