# 盯盘推送规则

更新时间：2026-07-20

## 通用盯盘币同结构去重

这条规则不只针对 AKE。后续所有放入“思路分析 / 重点盯盘”的币，都走同一套 Lark 推送逻辑。

### 核心原则

- 页面和数据库可以每 5 分钟正常刷新；
- Lark 只在出现新结构、新证据、新方向时推送；
- 如果方向、价格区间、判断剧本都没变，价格、持仓、CVD、盘口墙只是小幅抖动，不重复打扰；
- 机会来了仍然要立刻提醒，不设置机械冷却时间。

### 什么算新信息

- 方向或剧本改变；
- 价格较上次推送变化约 `6%` 以上；
- Binance 基差变化约 `0.5%` 以上；
- 资金费变号，或资金费大幅变化；
- 持仓较上次推送变化约 `25%` 以上；
- 盘口墙规模变化约 `80%` 以上；
- 30M / 1H / 2H 至少两个周期的 CVD 方向发生改变。

### 目标

机会来了立刻叫醒；没有新证据时安静盯盘。避免同一个结构、同一个判断、相近数值每 5 分钟重复推送，把真正重要的新变化淹没。
